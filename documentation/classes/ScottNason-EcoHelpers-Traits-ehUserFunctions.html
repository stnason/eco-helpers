ehUserFunctions

ehUserFunctions.php : 27

ecoHelpers functions for the Users Model.

This trait provides all of the functions related to managing and answering granular/ specific questions about a users roles.

Because this trait is added to the ehUser model (which User should extend) -- it provide the ability to use the Auth() helper like this: Auth()->user()->isAdmin($user_id) Auth()->user()->getUserRoles($user_id)
