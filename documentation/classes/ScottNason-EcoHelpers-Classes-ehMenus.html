ehMenus.php : 21

Responsible for building out the menuing system for the standard header.

Can use a Pages table for more complex systems or just the array in the eco-menus.php config file.

Note: This requires the Pages table in order to work. Note: A TOP LEVEL (Module) is a page with a 0 parent_id and menu_item = true.
