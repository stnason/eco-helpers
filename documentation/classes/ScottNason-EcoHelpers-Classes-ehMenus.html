ehMenus
in package
  • Application

ehMenus.php : 23

ehMenus is responsible for building out the menu-ing system data for the standard nav header and those used in the Menus/Pages entry screens.

It uses (and depends on) the pages table.

!! IMPORTANT !! - This IS NOT PART OF THE BACK-END SECURITY CHECK. SECURITY IS HANDLED THROUGH THE check_permissions MIDDLEWARE. - ehMenus only controls what SHOWS up on the user's personal menu tree -- not the "access" to it.
