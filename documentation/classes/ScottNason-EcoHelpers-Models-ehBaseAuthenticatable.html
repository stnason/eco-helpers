ehBaseAuthenticatable.php : 26

The base model for the ehUser model to extend; this obviously extends the underlying Laravel Authenticatable.

Pull together custom functionality that we need for all models (like the ability to store and retrieve the label names; format data for saving, manage custom time stamps, etc.)

See also ehBaseModel that needs to be kept in sync with the functionality provided here.
