Layout class.

Handles all standard page/form area display control interaction to the base template.

  • banner, name, icon, description, linkbar, heading, flash, attention, option-block, login

  • NOTE: that the header and footer areas are now completely independent and controlled by their own template files.

public static __callStatic( $method[, mixed $passed_value = null ]) : mixed

Where "Name" refers to one of the display areas defined in self::$defined_areas.

Note: that below here, we built out all the needed setters. Note: that this magic method could be used without the setters, but they are needed for IDE type-ahead help.

USAGE: setName('some value') - set the text in the display area. setName(true/false) - turn the display area on or off. setName() - same as setName(true);
